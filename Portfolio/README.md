# ⟐ Antigravity Portfolio Workspace

Welcome to the **Antigravity** futuristic portfolio workspace. This project is a state-of-the-art showcase for visionaries.

## 🚀 Getting Started

### 1. Enable Python Environment
To fully utilize the workspace features, ensure you have Python installed. It is recommended to create a virtual environment:

```powershell
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Use the Portfolio Manager
The workspace includes a powerful CLI tool `main.py` to help you manage the project.

- **Check Project Status**:
  ```powershell
  python main.py status
  ```
- **Preview Portfolio**:
  ```powershell
  python main.py serve
  ```
- **Project Info**:
  ```powershell
  python main.py info
  ```

## 📂 Project Structure

- `index.html`: The main portfolio website.
- `main.py`: Workspace management CLI.
- `assets/`: Team images and static resources.
- `requirements.txt`: Python dependencies.

## 🎨 Design Philosophy
Antigravity is built with **Rich Aesthetics** in mind, featuring:
- **Glassmorphism**: Sleek, translucent cards with backdrop blurs.
- **Dynamic Animations**: Floating elements and starfield backgrounds.
- **Neon Accents**: Vibrant purple, cyan, and pink color palettes.
